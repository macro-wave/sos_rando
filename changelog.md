# RELEASES

## Version 0.0.10 : 4/11/2026

- Fixed a bug that prevented vanilla from loading a save
- Fixed small logic errors
- Fixed vanilla NG+ to treat conches and fish as unlocked
- Changed order of some items to better match location
- Changed a lot of sprite and icon art
- Added items to track boosting, sailing, and flying abilites, and player level
- Added logic to track availability of ingredients for Chomper
- Added new, sleeker versions of the maps
- Added some unused malevolent spectacles
- Removed item hosting from vanilla tracker
- Removed ability rules in favor of ability items
- Removed the Training Weapons check that was eliminated from the prologue in DoE


## Version 0.0.9 : 02/23/2026

- Fix a logic error in the Duke Aventry Sidequest
- Fix an error where cookie jar was not required for reclamation
- Fix logic errors in Mirth checks
- Changed to entire structure of bosses; now logic considers whether you _can_ beat a boss, regardless of whether you have yet
- Changed to treat cypher as a shuffled item
- Changed the background to a better contrasting color so items can be seen in disabled state
- Added new option: Docarri ruin prism towers may be considered already raised _or_ already activated, to skip some item requirements
- Added logic trick option for accessing Romaya's secret passage and Oasis without seeing their respective hints first
